# UI/UX (Google Stitch) Agent Skill

## Purpose
Maintain a premium, high-end "Glassmorphism" design aesthetic inspired by modern Google design loops and NobleMind brand identity.

## Core Directives
1. **Vibrant Palettes**: Avoid flat colors. Use HSL-based gradients and deep charcoal/navy for dark mode.
2. **Glassmorphism**: Use `backdrop-blur`, subtle borders (`border-white/10`), and semi-transparent backgrounds.
3. **Micro-animations**: Integrate Framer Motion for entrance animations, hover scales, and smooth page transitions.
4. **Typography**: Use premium Google Fonts (Outfit, Inter) with consistent line heights and letter spacing.
5. **Consistency**: Follow the established design tokens in `tailwind.config.ts`.
6. **Responsiveness**: Ensure visually stunning layouts across all breakpoints (Mobile-First).

## Style Checklist
- [ ] Glassmorphism applied to cards and nav
- [ ] Hover states for all interactive elements
- [ ] Smooth entrance animations (Framer Motion)
- [ ] Accessibility (Contrast ratios > 4.5:1)
